# Триdent — сайт (полагоджена версія)

Односторінковий сайт стоматології. Ця версія — перероблена за результатами аудиту:
вага впала з **1,87 МБ до ~104 КБ** HTML, форма більше не втрачає заявки, додано
доступність, SEO, згоду на обробку даних, месенджери та блок цін.

## Структура
```
trident/
├── index.html            ← сам сайт
├── images/               ← оптимізовані зображення (WebP)
│   ├── logo.webp
│   ├── doc-*.webp        ← фото лікарів
│   └── img-*.jpg         ← сюди покладіть власні фото клініки (див. нижче)
├── telegram-backend.gs   ← backend форми → Telegram (Google Apps Script)
└── README.md
```
Заливайте на хостинг **усю папку разом** (index.html + images поруч).

## Що вже зроблено
- **Форма більше не бреше.** Прибрано фейкове «Заявку надіслано». Додано honeypot
  (антиспам) і обов'язкову згоду на обробку даних. Поки backend не підключено —
  заявка **не втрачається**: відкривається SMS адміністратору з готовим текстом.
- **Зображення локалізовано й стиснуто.** Фото лікарів і лого винесені з HTML у
  WebP-файли (менше й кешуються). Фото клініки тепер тягнуться з локальної папки
  `images/`, а якщо файлу ще немає — автоматично підвантажуються зі старого джерела
  (сайт не ламається).
- **Швидкість:** lazy-load, decoding=async, забрано 1,2 МБ base64 з HTML.
- **Доступність:** skip-link, `<main>`, aria-expanded на меню й FAQ, підписи полів
  (label for/id), пастка фокуса в модалках, повернення фокуса, підтримка
  `prefers-reduced-motion`, видимий фокус із клавіатури.
- **SEO:** активні canonical та og:url, розмітка FAQPage.
- **Право:** чекбокс згоди, модалка «Політика конфіденційності», cookie-банер.
- **Конверсія:** кнопки Viber/Telegram (FAB + контакти), блок орієнтовних цін.
- **Аналітика:** готові (закоментовані) блоки GA4 і Meta Pixel; форма вже надсилає
  подію ліда, щойно ви їх увімкнете.

## Що треба зробити ВАМ (обов'язково перед публікацією)
1. **Домен.** У `index.html` замініть `REPLACE-WITH-YOUR-DOMAIN` (2 місця: canonical,
   og:url) на ваш реальний домен.
2. **Telegram-форма.** Відкрийте `telegram-backend.gs`, виконайте інструкцію вгорі
   файлу, отримайте URL і вставте його в `index.html` → `TRIDENT_CONFIG.formEndpoint`.
3. **Telegram-кнопки.** У `index.html` замініть `REPLACE-TG-USERNAME` на ваш нік у
   Telegram (напр. `trident_nezhin`). Viber уже працює за номером.
4. **Ціни.** У розділі «Орієнтовні ціни» замініть `___ грн` на реальні значення.
5. **Фото клініки (рекомендовано).** Покладіть власні знімки в `images/` під іменами:
   `img-reception.jpg`, `img-cabinet.jpg`, `img-interior.jpg`, `img-entrance.jpg`,
   `img-building.jpg`. Бажано WebP/JPG шириною ~1200 px, стиснуті. Доки їх немає —
   сайт показує старі фото з Google.
6. **Аналітика (рекомендовано).** Розкоментуйте блоки GA4/Meta Pixel у `<head>` і
   впишіть свої ID.

## Локальна перевірка
Відкрийте `index.html` у браузері подвійним кліком — усе працює офлайн, окрім
надсилання у Telegram (для нього потрібен розгорнутий backend).
